## 2. INSTALLATION GUIDE

### For Raters (Mobile App)

#### Step 1: Receive the APK File
- Coordinator will send you: `VAW_Assessment_Baggao.apk`
- Save it to your phone's Downloads folder

#### Step 2: Enable Installation from Unknown Sources
1. Open **Settings** on your Android phone
2. Go to **Security** or **Privacy**
3. Find **Install Unknown Apps** or **Unknown Sources**
4. Enable for **Files** or **My Files** app
5. Confirm the change

#### Step 3: Install the App
1. Open **Files** or **My Files** app
2. Navigate to **Downloads** folder
3. Tap on `VAW_Assessment_Baggao.apk`
4. Tap **Install**
5. Wait for installation to complete (10-20 seconds)
6. Tap **Open** or **Done**

#### Step 4: Verify Installation
1. Go to your phone's home screen
2. Look for the VAW Assessment app icon
3. Tap to open
4. You should see the dashboard screen

**‚úÖ Installation complete!**

## 3. RATER INSTRUCTIONS

### Your Assignment
- You will assess **48 barangays**
- Each assessment has **7 steps**
- Takes approximately **15-20 minutes per barangay**
- Total time: 12-16 hours spread over several weeks

### Starting an Assessment

#### Step 1: Open the App
1. Tap the VAW Assessment icon on your home screen
2. Wait for dashboard to load (2-3 seconds)

#### Step 2: Select Barangay
1. You'll see a list of 48 barangays
2. Tap dropdown menu "Select Barangay to Assess"
3. Choose a barangay (alphabetically sorted)
4. Tap "Start Assessment" button

#### Step 3: Confirm Details
- Check that the date is correct
- Verify barangay name
- Tap "Next" to begin

### Completing the Assessment

#### The 7 Steps:

**Step 1: Barangay Selection**
- Select barangay name
- Confirm assessment date
- Review instructions

**Step 2: Establishment (Part 1)**
- Question 1.a: Ordinance/Executive Order
- Select ONE option
- Score: 0-5%

**Step 3: Establishment (Part 2)**
- Question 1.b: Trainings attended (check all that apply)
- Question 1.c: Location
- Question 1.d: Interview room
- Score: 0-15%

**Step 4: Resources (Furniture & Equipment)**
- Question 2.a: Furniture items (check all that apply)
- Question 2.b: Equipment items (check all that apply)
- Score: 0-10%

**Step 5: Resources (Monitoring Tools)**
- Question 2.c: Logbooks and forms (check all that apply)
- Question 2.d: Reference materials (check all that apply)
- Score: 0-10%

**Step 6: Policies, Plans, and Budget**
- Question 3.a: AIP
- Question 3.b: Certificate
- Question 3.c: GAD Plan
- Question 3.d: BDP
- Score: 0-20%

**Step 7: Accomplishments**
- Question 4.a: Annual report
- Question 4.b: Quarterly reports (select number)
- Question 4.c: Database
- Question 4.d: SOBA inclusion
- Score: 0-40%

#### Navigation Tips:
- **Next Button**: Move to next step
- **Previous Button**: Go back to previous step
- **Score Display**: Always visible at top of each section
- **Auto-Save**: Data saves automatically after each step

### Reviewing Your Assessment

#### Summary Screen:
After Step 7, you'll see:
- Final total score (out of 100)
- Breakdown by section
- Barangay name and date
- Your name as assessor

#### If You Need to Change Answers:
1. Tap "‚Üê Edit Answers" button
2. Go back to any step
3. Make changes
4. Navigate forward to summary again

### Saving the Assessment

#### Final Save:
1. Review the summary screen carefully
2. Check all scores are correct
3. Tap "üíæ Save Assessment" button
4. Wait for confirmation message
5. You'll return to dashboard

#### Verification:

## 5. TROUBLESHOOTING

### For Raters

#### Problem: App won't install
**Solutions:**
- Check: Is "Unknown Sources" enabled?
- Try: Restart phone and try again
- Check: Is there enough storage? (Need at least 50MB)
- Try: Delete old downloads and try again

#### Problem: App crashes when opening
**Solutions:**
- Force stop the app (Settings ‚Üí Apps ‚Üí VAW Assessment ‚Üí Force Stop)
- Clear app cache (Settings ‚Üí Apps ‚Üí VAW Assessment ‚Üí Clear Cache)
- Restart phone
- Reinstall app if needed

#### Problem: Can't find barangay in dropdown
**Solutions:**
- Scroll through the full list (48 barangays)
- Check spelling (use official names)
- Try searching alphabetically

#### Problem: Score doesn't seem right
**Solutions:**
- Review each section's answers
- Check if all checkboxes are correct
- Remember: Each item has specific point values
- Use "Previous" button to review and fix

#### Problem: "Save Assessment" button doesn't work
**Solutions:**
- Make sure you've completed all 7 steps
- Check if there's an error message
- Try going back and forward again
- Force close and reopen app
- Your data is auto-saved, so you won't lose progress

#### Problem: Lost data after phone restart
**Solutions:**
- Open app - data should still be there (saved locally)
- Check dashboard for completed assessments
- If really gone, check Downloads folder for backup JSON files
- Contact coordinator for help

#### Problem: Can't submit data (no internet)
**Solutions:**
- Save data first (it's on your phone)
- Wait until you have WiFi
- Or download JSON file to phone
- Send via email when online
- Or bring phone to coordinator for manual transfer

### For Coordinator

#### Problem: Rater says they can't install app
**Solutions:**
- Walk them through Settings ‚Üí Security step-by-step
- Try different file transfer method (Bluetooth, USB)
- Visit them in person if needed
- Test on their specific phone model

#### Problem: Received corrupted JSON file
**Solutions:**
- Ask rater to resend
- Try opening in text editor (Notepad++)
- Check file size (should be ~100-500KB)
- If truly corrupted, rater may need to resubmit from phone

#### Problem: Consolidator shows error loading files
**Solutions:**
- Check: Are files actually .json format?
- Try: Load files one at a time
- Verify: Files are not empty (>1KB)
- Try: Different browser (Chrome, Firefox, Edge)

#### Problem: Excel has wrong totals
**Solutions:**
- Check source JSON files manually
- Verify all 10 files were loaded
- Reprocess with fresh browser session
- Contact technical support

#### Problem: Can't open Excel file
**Solutions:**
- Need Microsoft Excel 2010 or later
- Or use Google Sheets (upload the file)
- Or use LibreOffice Calc (free)
- Check file actually downloaded completely

## 8. FREQUENTLY ASKED QUESTIONS

### For Raters:

**Q: How long does each assessment take?**
A: Approximately 15-20 minutes per barangay.

**Q: Can I do multiple barangays in one day?**
A: Yes, as many as you want. Data saves automatically for each.

**Q: What if I make a mistake?**
A: You can edit any assessment before final submission. Just select the barangay again.

**Q: Do I need internet?**
A: No, for assessments. Yes, only for final email submission.

**Q: What if my phone dies during assessment?**
A: Data auto-saves after each step. When you reopen, you can continue.

**Q: Can I uninstall the app after submitting?**
A: Wait until coordinator confirms receipt of your data first.

### For Coordinator:

**Q: What if a rater loses their data?**
A: Ask them to check phone backups. If truly lost, they must redo those barangays.

**Q: What if I only receive 9 out of 10 files?**
A: Follow up with missing rater. You can still generate report with 9, but note the gap.

**Q: Can I edit the Excel report?**
A: Yes! Add your analysis, format for presentation, add charts, etc.

**Q: What if scores seem suspiciously high/low?**
A: Check the raw data sheet. You can spot-check and contact rater if needed.

**Q: How do I present this to the Chairman?**
A: Prepare PowerPoint from Excel data. Focus on key findings and recommendations.
